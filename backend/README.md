# Backend Vision Reference

This folder contains the structured vision and strategy for karsenkoltun.com, extracted from the Website Design Vision Document (February 2026).

## Files

- `vision.md` - Core vision, principles, and design direction
- `pages.md` - Detailed breakdown of every page (homepage, sell, buy, etc.)
- `navigation.md` - Navigation structure and menu hierarchy
- `integrations.md` - GoHighLevel, IDX/MLS, and technical integration notes
- `design-system.md` - Color palette, typography, motion, and photography direction
- `sitemap.md` - Full site map and route structure
